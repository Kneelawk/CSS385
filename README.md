## Class Requirements

### Persistent Background Music

The game has persistent background music when in a level (created by [Lynndova](https://lynndova.bandcamp.com/)).

### Triggered Sound Effects

When something is on top of a button, the button depresses and makes a clicking sound.

### Triggered Music

In Level 1, once you get beyond the last door, the music changes to a track made by me.

### Triggered Video

I was not able to incorporate full video playback into my program. However, I was able to incorporate a triggered animation, which the teacher seemed to indicate would fill this requirement.
