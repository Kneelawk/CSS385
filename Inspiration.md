# Inspirations

In this class, I am drawing inspiration from my favorite games: Reassembly, Factorio, RoboCraft, and Space Engineers. Reassembly is a game where you build a 2d spaceship fleet and conquer other space factions with your space ships. Factorio is a game about building factories, the game's listed objective is to launch a rocket using resources collected and built by the player, but the game's real objective is simply to create an ever-expanding factory to create tools to allow for a larger and more advanced factory. RoboCraft is a MOBA (Multiplayer Online Battle Arena) game where players build their own robots outside of battles and then use them to fight other players' robots in the battle, competing to achive various goals and capture various territory. Space Engineers is a game about building, flying, and battling space-ships in a 3D universe. I am drawing inspiration from these games in particular because I have always wanted to play a game that combines the features of these games.
