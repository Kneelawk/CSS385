# Brainstorm

There is a game I have wanted to build for a long time that combines the features of some of my favorite factory and MOBA games. This game is about battling other teams of space ships for resources and to ultimately capture their bases. I want this game to be just as much about building and designing the space ships as using them to fight other teams' space ships. I will break the game's components down into a number of bullet points:

* The game will be 2D.
* The game is a MOBA (Multiplayer Online Battle Arena) thought the game may also have a survival or collaborative mode.
* The game is about building and battling space ships.
* The the MOBA portion of the game will consist of space ships collecting resources, assembling space ships designed before the battles using collected resources, and fighting tother teams' space ships for resources and to conquer their bases.
* Each space ship will be a factory, at lest to some extent.
    * This means that the position that space ship components are placed at will matter and some components soul purpose will be to support other components.
    * I want there to be a significant degree of creativity allowed when designing functional space ships.
