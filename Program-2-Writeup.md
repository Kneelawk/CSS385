# Program 2 Interaction Model

The Game's interaction model is the player using the keyboard WASD keys and mouse to direct a ball around a physics world. The objective of the interaction is to engage with various physics mechanics within the world and to complete puzzles.

## How this Interaction Model can be used

This interaction model is the core mechanic of the game. This means that the primary way that the player is intended to interact with the game world is by rolling the ball. First, this interaction model can be used to navigate around and explore each level. Furthermore, since the player-controled ball is a physics object itself, this interaction model can also be used to complete various physics-based puzzles.L
